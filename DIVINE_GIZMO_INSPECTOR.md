# Божественный Гизмо и Инспектор — Готово! ✨

## Что было внедрено

### 1. Божественный Гизмо (Divine Gizmo)
Полноценная 3D система манипуляторов в стиле Unreal/Godot/Unity:

#### Компоненты гизмо:
- **Ствол (Cylinder)** — тонкий длинный цилиндр для каждой оси
- **Стрелка (Cone)** — конус на конце для операции Move
- **Кубик (Cube)** — маленький куб посередине для операции Scale
- **Кольцо (Torus)** — торус вокруг каждой оси для операции Rotate

#### Особенности:
- Три оси (X, Y, Z) с цветовой кодировкой:
  - X — Красный (0.9, 0.2, 0.2)
  - Y — Зеленый (0.2, 0.9, 0.2)
  - Z — Синий (0.2, 0.2, 0.9)
- Адаптивный размер — гизмо масштабируется в зависимости от расстояния камеры
- Рентген-режим — гизмо всегда видно поверх всех объектов (gl.disable(gl.DEPTH_TEST))
- Правильная ориентация — каждая ось корректно повернута через кватернионы

#### Файлы:
- `vscode/src/vs/workbench/contrib/voidSceneEditor/browser/threeViewport.ts`
  - Добавлена функция `m4Copy()` для копирования матриц
  - Полностью переписан метод `renderGizmo()` с отрисовкой всех компонентов

### 2. Инспектор Свойств (Inspector View)
Правая панель для редактирования свойств выделенного объекта в реальном времени:

#### Возможности:
- **Редактирование Transform:**
  - Translation (Vec3) — позиция объекта
  - Scale (Vec3) — масштаб объекта
  - Rotation (Vec4) — кватернион вращения
  
- **Редактирование Material:**
  - Color (Color Picker) — цвет материала
  - Metallic (Float) — металличность
  - Roughness (Float) — шероховатость
  
- **Редактирование PointLight:**
  - Color (Color Picker) — цвет света
  - Intensity (Float) — интенсивность
  - Range (Float) — радиус действия
  
- **Редактирование Mesh:**
  - Radius, Size, Height, Tube — параметры геометрии

#### Особенности:
- Автоматическое обновление при выборе объекта
- Изменения сразу сохраняются в .vecn файл
- Изменения мгновенно отображаются во вьюпорте
- Поддержка иерархии — рекурсивный поиск объектов
- Красивый UI в стиле VS Code Dark Theme

#### Файлы:
- `vscode/src/vs/workbench/contrib/voidSceneEditor/browser/inspectorView.ts` (НОВЫЙ)
  - Полная реализация инспектора
  - UI компоненты для всех типов свойств
  - Интеграция с VecnParser для сериализации

## Поток данных

```
User Click → Viewport (Picking) → onEntityPicked Event
                                        ↓
                                  Inspector.selectEntity()
                                        ↓
                                  Render Properties
                                        ↓
User Edit Input → onChange → Update Entity → VecnParser.serialize()
                                        ↓
                                  TextModel.setValue()
                                        ↓
                                  Model Change Event
                                        ↓
                                  Viewport.updateScene()
                                        ↓
                                  Re-render with new data
```

## Как использовать

1. Откройте .vecn файл
2. Переключитесь в режим "3D Scene" через тулбар
3. Кликните на объект во вьюпорте
4. Справа появится инспектор с его свойствами
5. Измените любое значение (позиция, цвет, масштаб)
6. Изменения мгновенно применятся к объекту и сохранятся в файл

## Технические детали

### Гизмо
- Использует существующую геометрию: `cylinderGeo`, `coneGeo`, `cubeGeo`, `torusGeo`
- Рисуется через `flatProgram` (простой цветной шейдер)
- Адаптивный масштаб: `baseScale = max(0.1, distance * 0.18)`
- Отключен depth test для рентген-эффекта

### Инспектор
- Dependency Injection через `@ITextModelService`
- Слушает `vecnSceneBus` для синхронизации
- Использует `TextModelService.createModelReference()` для записи
- Рекурсивный поиск объектов в иерархии
