# Введение

**Void Engine** — это форк VS Code с интегрированным игровым движком на базе **Bevy ECS (Rust)**. Главная особенность — встроенный редактор сцен с 3D WebGL2 viewport для редактирования файлов `.vecn` (RON-подобный формат).

## Ключевые особенности

| Возможность | Описание |
|-------------|----------|
| **3D Viewport** | Pure WebGL2 рендеринг (без Three.js) |
| **Scene Editor** | Иерархия и инспектор в стиле Godot |
| **50+ типов узлов** | 2D, 3D, физика, аудио, анимация, навигация |
| **Процедурное небо** | Шейдер в стиле UE5 с атмосферным рассеянием |
| **Гизмос трансформации** | Инструменты перемещения, вращения, масштаба |
| **ECS архитектура** | Bevy-совместимая система компонентов |

## Bevy Plugin

Void Engine включает готовый Bevy плагин для загрузки `.vecn` файлов:

```rust
use bevy::prelude::*;
use void_scene_loader::VoidSceneLoaderPlugin;

fn main() {
    App::new()
        .add_plugins(DefaultPlugins)
        .add_plugins(VoidSceneLoaderPlugin) // Автозагрузка main.vecn
        .run();
}
```

# Формат файлов .vecn

`.vecn` использует RON-подобный синтаксис (Rusty Object Notation) для описания иерархии сцен.

## Базовая структура

```ron
VoidScene(
    version: "1.0",
    mode: Scene3D,
    
    entities: [
        (
            id: "entity-uuid",
            name: "Player",
            visible: true,
            components: [
                Transform(
                    translation: (0, 1, 0),
                    rotation: (0, 0, 0, 1),
                    scale: (1, 1, 1),
                ),
                Mesh( shape: Cube(size: 1) ),
                Material(
                    color: (0.6, 0.6, 0.6, 1.0),
                    metallic: 0,
                    roughness: 0.8,
                ),
            ],
            children: [],
        ),
    ],
    
    resources: [],
)
```

## Типы данных

| Тип | Пример | Описание |
|-----|--------|----------|
| `String` | `"Player"` | Строка в кавычках |
| `Number` | `1.0`, `42` | Числа с/без точки |
| `Boolean` | `true`, `false` | Булевы значения |
| `Tuple` | `(1, 2, 3)` | Кортеж (вектор) |
| `Enum` | `Scene3D`, `Cube(size: 1)` | Перечисления |
| `List` | `[...]` | Массив элементов |

# Базовые компоненты

## Transform (рабочий)

**Описание:** Позиция, вращение и масштаб объекта в 3D пространстве. Это обязательный компонент для любой 3D сущности.

### Связь с Bevy

```
┌─────────────────┐         ┌─────────────────┐
│  .vecn          │         │  Bevy           │
├─────────────────┤         ├─────────────────┤
│ Transform       │   ───►  │ Transform       │
│  - translation  │         │  - translation  │
│  - rotation     │         │  - rotation     │
│  - scale        │         │  - scale        │
└─────────────────┘         └─────────────────┘
```

### Свойства

| Свойство | Тип | По умолчанию | Описание |
|----------|-----|--------------|----------|
| `translation` | `(f32, f32, f32)` | `(0, 0, 0)` | Позиция в мировых координатах (X, Y, Z) |
| `rotation` | `(f32, f32, f32, f32)` | `(0, 0, 0, 1)` | Кватернион вращения (x, y, z, w) |
| `scale` | `(f32, f32, f32)` | `(1, 1, 1)` | Масштаб по осям (X, Y, Z) |

### Инспектор

| Поле | Виджет | Описание |
|------|--------|----------|
| Translation X | Slider + Input | Красный цвет оси |
| Translation Y | Slider + Input | Зелёный цвет оси |
| Translation Z | Slider + Input | Синий цвет оси |
| Rotation | Euler Angles | Конвертируется из кватерниона |
| Scale X/Y/Z | Slider + Input | С возможностью блокировки пропорций |

### Пример в .vecn

```ron
Transform(
    translation: (5.0, 2.0, -3.0),
    rotation: (0.0, 0.707, 0.0, 0.707), // 90° по Y
    scale: (1.0, 2.0, 1.0),
)
```

# Меши и материалы

## Mesh (рабочий)

**Описание:** Геометрия объекта. Определяет форму 3D модели.

### Доступные формы

| Форма | Параметры | Пример |
|-------|-----------|--------|
| **Cube** | `size: f32` | `Cube(size: 1)` |
| **Sphere** | `radius: f32` | `Sphere(radius: 0.5)` |
| **Plane** | `size: f32` | `Plane(size: 10)` |
| **Cylinder** | `radius, height: f32` | `Cylinder(radius: 0.5, height: 1)` |
| **Cone** | `radius, height: f32` | `Cone(radius: 0.5, height: 1)` |
| **Torus** | `radius, tube: f32` | `Torus(radius: 1, tube: 0.3)` |
| **Capsule** | `radius, height: f32` | `Capsule(radius: 0.3, height: 1)` |

### Связь с Bevy

```
┌─────────────────┐         ┌─────────────────┐
│  .vecn Mesh     │         │  Bevy           │
├─────────────────┤         ├─────────────────┤
│ Cube(size: 1)   │   ───►  │ Cuboid::new()   │
│ Sphere(r: 0.5)  │   ───►  │ Sphere::new()   │
│ Plane(size: 10) │   ───►  │ Plane3d         │
│ Cylinder(...)   │   ───►  │ Cylinder::new() │
│ Cone(...)       │   ───►  │ Cone            │
│ Torus(...)      │   ───►  │ Torus           │
│ Capsule(...)    │   ───►  │ Capsule3d       │
└─────────────────┘         └─────────────────┘
```

### Инспектор

| Поле | Виджет | Описание |
|------|--------|----------|
| Shape | Dropdown | Выбор типа формы |
| Size/Radius/etc. | Number Input | Параметры формы |

## Material (рабочий)

**Описание:** Визуальные свойства поверхности объекта.

### Свойства

| Свойство | Тип | По умолчанию | Диапазон | Описание |
|----------|-----|--------------|----------|----------|
| `color` | `(f32, f32, f32, f32)` | `(0.6, 0.6, 0.6, 1.0)` | 0.0-1.0 | RGBA цвет |
| `metallic` | `f32` | `0` | 0.0-1.0 | Металличность поверхности |
| `roughness` | `f32` | `0.8` | 0.0-1.0 | Шероховатость (0 = зеркало) |

### Визуализация свойств

```
metallic: 0                    metallic: 1
    │                              │
    ▼                              ▼
┌─────────┐                  ┌─────────┐
│ Пластик │                  │ Металл  │
│ Дерево  │                  │ Золото  │
│ Камень  │                  │ Хром    │
└─────────┘                  └─────────┘

roughness: 0                  roughness: 1
    │                              │
    ▼                              ▼
┌─────────┐                  ┌─────────┐
│ Зеркало │                  │ Матовый │
│ Глянец  │                  │ Шершавый│
│ Полир.  │                  │ Грубый  │
└─────────┘                  └─────────┘
```

### Инспектор

| Поле | Виджет | Описание |
|------|--------|----------|
| Color | Color Picker | RGBA выбор цвета |
| Metallic | Slider | От "неметалл" до "металл" |
| Roughness | Slider | От "гладкий" до "шероховатый" |

### Пример материалов

```ron
// Пластик (красный, глянцевый)
Material(
    color: (0.9, 0.2, 0.2, 1.0),
    metallic: 0.0,
    roughness: 0.3,
)

// Хром (зеркальный)
Material(
    color: (0.8, 0.8, 0.85, 1.0),
    metallic: 1.0,
    roughness: 0.1,
)

// Золото
Material(
    color: (1.0, 0.84, 0.0, 1.0),
    metallic: 1.0,
    roughness: 0.4,
)
```

# Освещение

## PointLight (рабочий)

**Описание:** Точечный источник света, светит во всех направлениях из одной точки.

```
        ╲   │   ╱
         ╲  │  ╱
          ╲ │ ╱
    ──────●──────
          ╱ │ ╲
         ╱  │  ╲
        ╱   │   ╲
        
      PointLight
      (светит везде)
```

### Свойства

| Свойство | Тип | По умолчанию | Описание |
|----------|-----|--------------|----------|
| `color` | `(f32, f32, f32)` | `(1, 1, 1)` | RGB цвет света |
| `intensity` | `f32` | `1000` | Интенсивность (люмены) |
| `range` | `f32` | `10` | Дальность света в метрах |

### Инспектор

| Поле | Виджет | Описание |
|------|--------|----------|
| Color | Color Picker | Цвет света (без alpha) |
| Intensity | Slider + Input | Яркость источника |
| Range | Slider + Input | Радиус действия |

### Связь с Bevy

```rust
// .vecn → Bevy
commands.spawn((
    PointLight {
        color: Color::srgb(color.0, color.1, color.2),
        intensity,
        range,
        shadows_enabled: true,
        ..default()
    },
    transform,
));
```

# Физика 3D

> **ВНИМАНИЕ:** Физика в Bevy требует интеграции с `bevy_rapier3d` или `bevy_xpbd`. 
> В void_scene_loader компоненты физики являются заглушками.

## CharacterBody (заглушка)

**Описание:** Тело, управляемое игроком. Не подвержено физике, но может сталкиваться.

### Свойства

| Свойство | Тип | По умолчанию | Описание |
|----------|-----|--------------|----------|
| `mass` | `f32` | `1.0` | Масса в кг |
| `gravity_scale` | `f32` | `1.0` | Множитель гравитации |
| `lock_rotation` | `bool` | `false` | Заблокировать вращение |

### Инспектор

| Поле | Виджет | Описание |
|------|--------|----------|
| Mass | Number Input | Масса персонажа |
| Gravity Scale | Slider | 0 = нет гравитации, 1 = нормальная |
| Lock Rotation | Checkbox | Отключить вращение от столкновений |

---

## RigidBody (заглушка)

**Описание:** Физическое тело, подверженное силам и столкновениям.

### Свойства

| Свойство | Тип | По умолчанию | Описание |
|----------|-----|--------------|----------|
| `mass` | `f32` | `1.0` | Масса в кг |
| `gravity_scale` | `f32` | `1.0` | Множитель гравитации |
| `linear_damping` | `f32` | `0.0` | Линейное затухание |
| `angular_damping` | `f32` | `0.0` | Угловое затухание |
| `lock_rotation_x` | `bool` | `false` | Блокировка вращения X |
| `lock_rotation_y` | `bool` | `false` | Блокировка вращения Y |
| `lock_rotation_z` | `bool` | `false` | Блокировка вращения Z |

# Быстрый старт

## Создание первой сцены

1. Создайте файл `main.vecn` в папке `assets/scenes/`
2. Добавьте базовую структуру:

```ron
VoidScene(
    version: "1.0",
    mode: Scene3D,
    entities: [
        // Камера
        (
            id: "camera",
            name: "Camera",
            visible: true,
            components: [
                Transform(
                    translation: (0, 5, 10),
                    rotation: (-0.1, 0, 0, 0.995),
                    scale: (1, 1, 1),
                ),
                Camera(fov: 60, near: 0.1, far: 1000),
            ],
            children: [],
        ),
        // Свет
        (
            id: "sun",
            name: "Sun",
            visible: true,
            components: [
                Transform(
                    translation: (10, 20, 10),
                    rotation: (0, 0, 0, 1),
                    scale: (1, 1, 1),
                ),
                DirectionalLight(
                    color: (1.0, 0.95, 0.9),
                    illuminance: 1.0,
                ),
            ],
            children: [],
        ),
        // Куб
        (
            id: "cube",
            name: "Cube",
            visible: true,
            components: [
                Transform(
                    translation: (0, 0.5, 0),
                    rotation: (0, 0, 0, 1),
                    scale: (1, 1, 1),
                ),
                Mesh(shape: Cube(size: 1)),
                Material(
                    color: (0.8, 0.3, 0.3, 1),
                    metallic: 0.2,
                    roughness: 0.6,
                ),
            ],
            children: [],
        ),
    ],
    resources: [],
)
```

3. Запустите Bevy приложение с VoidSceneLoaderPlugin

# Заключение

Void Engine находится в активной разработке. Базовый рендеринг (меши, материалы, освещение) работает стабильно. Физика, аудио и сложные системы реализованы как заглушки для сохранения структуры данных.

Следите за обновлениями в репозитории: https://github.com/StarryCod/Void-engine
