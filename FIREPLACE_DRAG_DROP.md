# Интерактивный камин с Drag & Drop

## Обзор
Камин компиляции теперь полностью интерактивен! Можно перетаскивать дрова мышкой прямо в огонь.

## ✅ Чистая реализация (v2.0)

### 1. Улучшенная текстура поленьев
- **Единая текстура** для дров в дровнице и в огне
- **Радиальные градиенты** имитируют сучки на дереве
- **Линейный градиент** создает объем (блик → середина → тень)
- **Круглые торцы** с кольцами дерева
- Дрова в огне чуть "жареные" (теплые оттенки)

#### Цветовая палитра
- Дровница: `#d3934b` → `#b06b32` → `#6d3b21`
- В огне: `#ffb86b` → `#d77a38` → `#6d2b16`
- Золотое: `#fff7c3` → `#ffd700` → `#b8860b`

### 2. Настоящий Drag & Drop
- **Тащится само полено**, а не клон
- Элемент переносится в `document.body` с `position: fixed`
- Реально следует за курсором через `mousemove`
- Offset сохраняется - хватаешь за край, тащится за край

#### Механика
1. `mousedown` на полене → переносим в body, делаем fixed
2. `mousemove` → обновляем `left/top` по курсору
3. `mouseup` → проверяем попадание в камин через `getBoundingClientRect()`
4. Если попал → создаем новое полено в точке сброса
5. Если промах → полено исчезает, дровница пополняется

### 3. Простое падение без телепортов
- **Автоматический бросок**: Полено сразу создается в случайной точке топки
- **Ручной бросок**: Полено создается точно там, где отпустил мышку
- **Анимация появления**: `translateY(-30px)` → `translateY(0)` за 200ms
- **Широкий разброс**: X: 70-240px, Y: 235-270px

### 4. Универсальный метод создания
```typescript
createLogInFire(backWall, x, y, fromDrag, forcedGolden?)
```
- Используется и для DnD, и для автоброска
- Ограничивает координаты границами топки
- Обрабатывает золотые поленья
- Добавляет в массив `_logs` для симуляции горения

## Технические детали

### CSS классы
- `.stored-log` - дрова в дровнице (с текстурой, `cursor: grab`)
- `.dragged-log` - перетаскиваемое бревно (`position: fixed`, белая рамка)
- `.wood-log` - дрова в огне (теплые оттенки)
- `.wood-log.golden` - золотое полено (сияющее)

### TypeScript методы
- `createStoredLog()` - создает полено с обработчиком DnD
- `onRackLogMouseDown()` - начало перетаскивания
- `onRackLogMouseMove()` - обновление позиции
- `onRackLogMouseUp()` - проверка попадания и создание
- `createLogInFire()` - универсальное создание полена в огне
- `spawnProjectileAndLand()` - автоматический бросок (упрощенный)

### Золотые дрова
- **Шанс при автоброске**: 10%
- **Шанс при ручном броске**: 15% (выше!)
- **Эффект**: Огонь x5, вспышка экрана, 15 секунд
- **Визуал**: Золотистый градиент, свечение

## Что исправлено

### Проблемы v1.0
- ❌ Полено телепортировалось после анимации
- ❌ Тащился клон, а не само полено
- ❌ Сложная траектория с параболой
- ❌ Разная текстура в дровнице и огне

### Решения v2.0
- ✅ Полено создается заново в точке приземления
- ✅ Тащится само полено через `position: fixed`
- ✅ Простое появление с анимацией `translateY`
- ✅ Единая текстура с радиальными градиентами

## Игровой процесс
1. Компиляция запускается → огонь горит
2. Дрова автоматически появляются в случайных местах
3. Можно вручную перетаскивать дрова для контроля
4. Шанс получить золотое бревно → мега-огонь!
5. Дрова сгорают → превращаются в пепел → исчезают
6. Дровница автоматически пополняется

## Советы
- Хватай дрова за край - они будут тащиться естественно
- Бросай в центр огня для лучшего эффекта
- При ручном броске шанс золота выше (15% vs 10%)
- Дровница бесконечная - не бойся промахиваться
