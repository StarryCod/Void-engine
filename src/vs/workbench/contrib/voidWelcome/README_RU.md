# VoidWelcome Project Manager - Изменения

## Обзор

VoidWelcome был преобразован из демонстрационной обложки с фейковыми проектами в **полноценный менеджер проектов VoidEngine**.

## Основные изменения

### 1. Работа с реальными проектами на диске
- **Путь к проектам:** `C:\Users\Starred\Documents\VoidEngine\Projects`
- Все проекты сканируются напрямую с диска
- Проекты хранятся только в localStorage как кэш (для pinned status)
- При каждой загрузке выполняется актуальное сканирование папки

### 2. Создание новых проектов

**Шаблоны:**
- **3D Scene** - 3D сцена с камерой, светом, кубом и сферой
- **2D Scene** - 2D сцена с камерой, спрайтом и текстом
- **Blank** - Пустой Bevy проект

**Структура создаваемого проекта:**
```
Projects/
  my-project/
    Cargo.toml          # Bevy 0.15 + void-scene-loader
    .gitignore
    .void-meta.json     # Метаданные проекта
    src/
      main.rs           # Точка входа
    assets/
      scenes/
        main.vecn       # Сцена VoidEngine
```

### 3. Редактирование проектов

**Редактируемые поля:**
- Имя проекта (обновляет Cargo.toml и slug)
- Описание (сохраняется в .void-meta.json)
- Автор (сохраняется в .void-meta.json)
- Версия (обновляет Cargo.toml)
- Категория (3D, 2D, Game, Web, Library, Tool)

### 4. Удаление проектов

**Важно:** Проекты удаляются **физически с диска**, а не только из списка!

### 5. Дублирование проектов

Копирует всю структуру проекта на диск с новым именем.

### 6. Запуск проектов

Открывает проект в новом окне VS Code.

## Технические детали

### Формат .void-meta.json

```json
{
  "name": "My Project",
  "slug": "my-project",
  "description": "Project description",
  "author": "Author Name",
  "template": "3d",
  "createdAt": 1234567890,
  "updatedAt": 1234567890
}
```

### Определение типа проекта

- **3D** - если в main.rs есть `camera_3d` или `Camera3d`
- **2D** - если в main.rs есть `camera_2d` или `Camera2d`
- **Game** - по умолчанию

### Зависимости

VoidWelcome.ts теперь использует:
- `IWorkspaceContextService` - для работы с workspace
- `IWorkbenchEnvironmentService` - для доступа к окружению
- `IHostService` - для открытия окон VS Code

## Стили (CSS)

Добавлены улучшения:
- Анимация иконок при наведении
- Свечение индикатора "disk" для реальных проектов
- Поддержка иконок 3D/2D

## Использование

1. **При запуске VS Code без открытой папки** - автоматически показывается VoidWelcome
2. **Создание проекта** - кликните на "New Project" или карточку с плюсом
3. **Открытие проекта** - двойной клик или кнопка "Open"
4. **Редактирование** - кнопка "Edit" или F2
5. **Удаление** - кнопка "Delete" (с подтверждением)

## Горячие клавиши

- `Ctrl+K` - Фокус на поиске
- `Enter` - Открыть выбранный проект
- `F2` - Редактировать выбранный проект
- `Delete` - Удалить выбранный проект
- `Escape` - Закрыть модальное окно / выйти из поиска
