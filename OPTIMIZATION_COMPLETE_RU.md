## Как протестировать

### 1. Перезагрузи окно
```
Ctrl+R
```

### 2. Открой .vecn файл
```
void_engine/assets/scenes/test_cube.vecn
```

### 3. Переключись в Scene3D
Нажми кнопку "3D" в тулбаре

### 4. Проверь производительность
1. Выбери объект (клик)
2. Перетащи гизмо (зелёная стрелка)
3. Должно быть плавно, без лагов
4. Inspector обновляется мгновенно
5. Консоль чистая (F12 → Console)

### 5. Проверь гизмо режимы
1. Нажми **W** → Стрелки перемещения
2. Нажми **E** → Кольца вращения
3. Нажми **R** → Кубики масштабирования
4. Всё должно работать плавно

### 6. Проверь автосохранение
1. Перетащи объект
2. Подожди 500мс
3. Открой файл test_cube.vecn
4. Координаты должны обновиться
